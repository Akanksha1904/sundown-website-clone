# Sundown Website Clone

A frontend clone of the Sundown website,
built to practice scroll animations and
interactive UI effects.

## Built with
- HTML5, CSS3, JavaScript
- Locomotive Scroll (smooth scrolling)
- Swiper.js (carousel/slider)

## Purpose
Learning project — animation and scroll
interaction practice.

## Note
This is a clone for educational purposes only.
Original design by Sundown.
